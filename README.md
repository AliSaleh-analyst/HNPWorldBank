# HNPWorldBank
This project features a high-end, interactive Power BI dashboard analyzing the World Bank’s Health, Nutrition, and Population (HNP) Statistics. The report tracks critical health indicators over a 10-year period (2013–2022), providing regional benchmarks and country-level drill-down capabilities.

"The dataset used in this project is the World Bank HNP Statistics (2.3+ GB). Due to size constraints, the raw data is not included in this repo. You can download the source file here: [https://databank.worldbank.org/source/health-nutrition-and-population-statistics]."
